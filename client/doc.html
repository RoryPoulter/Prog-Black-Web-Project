GET /

http://127.0.0.1:8080/

GET /ingredients

http://127.0.0.1:8080/ingredients

Returns JSON data with an array of all the ingredients in the drinks. Has no query parameters.

Example Request

curl http://127.0.0.1:8080/ingredients
